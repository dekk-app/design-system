# Avatar

An Avatar is a graphical representation of a user or the user's character or persona.

## Installation

```bash
yarn add @dekk-ui/avatar
```

## Usage

### With Image

In many cases we want to show a user-provided image as an avatar.  
Use the avatar similar to an `img` tag. Set the `src` and the `alt` text of the image.

```tsx
import { Avatar } from "@dekk-ui/avatar";

export const MyComponent = () => {
	<div>
		<Avatar
			src="https://cdn.fakercloud.com/avatars/yalozhkin_128.jpg"
			alt="Avatar of Max Wilson"
		/>
		<Avatar
			src="https://cdn.fakercloud.com/avatars/ashernatali_128.jpg"
			alt="Avatar of Alex Thomson"
		/>
		<Avatar
			src="https://cdn.fakercloud.com/avatars/mighty55_128.jpg"
			alt="Avatar of Cameron Smith"
		/>
	</div>;
};
```

### With Initials

If the user does not provide an image we can use their initials instead!  
In other cases we need to render text in an Avatar.

```tsx
import { Avatar } from "@dekk-ui/avatar";

export const MyComponent = () => {
	<div>
		<Avatar>MW</Avatar>
		<Avatar>AT</Avatar>
		<Avatar>CS</Avatar>
		<Avatar>+4</Avatar>
	</div>;
};
```
